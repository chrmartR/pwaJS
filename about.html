About the App

This path generator app uses mapping data from OpenStreetMap to generate looped paths for running, biking, hiking, or more. Paths can be generated using two different modes, a random circular path generator or a customizable looped path generator. The circular path generator takes a starting location and preferred path distance and creates a loop that roughly follows a circular shape. The customizable path generator will attempt to create a path that takes into account your own personal preferences such as the hilliness of the route, or the type of road or trail the path follows.
